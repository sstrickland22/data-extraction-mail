# Email Attachment Extractor

A Python tool to automatically extract attachments from emails. Supports multiple email providers and saves attachments locally for easy access.

---

## About

Email Attachment Extractor is a Python-based project that connects to your email account, scans for incoming messages with attachments, and downloads them to a specified local folder.  

It is ideal for automating workflows that require processing attachments from multiple emails.

---

## Features

- Extract attachments from Gmail, Outlook, and IMAP-compatible accounts.
- Automatically save attachments to a local folder.
- Filter emails by sender, subject, or date.
- Logs downloaded attachments to avoid duplicates.
